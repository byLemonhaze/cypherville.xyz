# Cypherville: The Dual Story

A narrative collection site for two linked Bitcoin Ordinals collections — *Cypherville* and *DeVille* — told as two sides of the same world.

Live: [cypherville.xyz](https://cypherville.xyz)

---

## What It Is

Cypherville and DeVille are companion collections inscribed on Bitcoin. This site frames them as a dual narrative — two factions, two aesthetics, one story. Visitors can explore each collection through a 3D carousel slideshow, read the history, and browse the tales tied to each piece.

The interface has no wallet integration, no marketplace links. It's built as an experience first.

---

## Structure

- **Landing** — dual entry point (Cypherville / DeVille)
- **Slideshow** — 3D carousel for browsing portraits
- **Grid** — full collection overview
- **Intro / History / Tales** — narrative context for each collection

---

## Stack

- Vanilla JS — custom 3D carousel (`Carousel3D.js`)
- Vite
- Cloudflare Pages + Functions (hosting)
